Данные учетной записи указаны в инструкции к сервису для каждого диспетчера
Какие есть минимальные требования к техническим средствам на рабочих местах диспетчера?
Требуется: дополнительный телефонный номер (для связи диспетчера с исполнителем), ПК (или ноутбук), принтер, средства диспетчеризации (диспетчерской связи)
